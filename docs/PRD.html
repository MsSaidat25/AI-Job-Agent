Data Strategy

Job Data Strategy

API availability assessment, tiered integration plan, and staffing agency coverage analysis.

Doors That Are CLOSED

LinkedIn API

Enterprise-only, 3-6 month approval, 6-figure annual commitment. Not accepting new partners.

Indeed Job Search API

Deprecated. Requires 10k+ daily page views. Sponsored Jobs API costs $3/CALL.

Glassdoor API

Developer program discontinued entirely.

ZipRecruiter ZipSearch

Shut down March 31, 2025.

Doors That Are OPEN
Tier 1 - Start Here

JSearch API (RapidAPI)

Google for Jobs aggregation. LinkedIn, Indeed, Glassdoor, ZipRecruiter data. Free: 200 req/mo. $25/mo: 10k req. $75/mo: 50k req.

Adzuna API

16+ countries, salary data, free tier.

Remotive API

Remote jobs, free (4 req/day).

Tier 2 - Scale Up

TheirStack

315k+ sources, ATS platforms, company enrichment. 90 EUR/mo.

SerpApi

Google Jobs with US Legal Shield. $25/mo.

Interactive compliance checklist. Click items to mark progress.

Classification: HIGH-RISK (Annex III, Section 4: Employment AI)
Deadline: August 2, 2026 (possible extension to Dec 2027)

  • ✓ Risk Management System Art. 9
  • ✓ Data Governance & Bias Testing Art. 10
  • ✓ Technical Documentation Art. 11
  • ✓ Automatic Event Logging, 6+ month retention Art. 12
  • ✓ Transparency to Users Art. 13
  • ✓ Human Oversight / Override capability Art. 14
  • ✓ Accuracy & Robustness Testing Art. 15
  • ✓ Quality Management System Art. 17
  • ✓ Conformity Self-Assessment Art. 43
  • ✓ EU Database Registration Art. 49
  • ✓ Fundamental Rights Impact Assessment Art. 27

GDPR Required Controls

  • ✓ Consent Management - explicit opt-in for AI processing Art. 6
  • ✓ Right to Explanation - show why jobs were recommended Art. 22
  • ✓ Data Portability API - export user data as JSON/CSV Art. 20
  • ✓ Right to Erasure API - full deletion within 30 days Art. 17
  • ✓ Data Minimization - collect only necessary data Art. 5

Risk Management

Risk Register

Key risks ranked by severity with mitigation strategies.

High

JSearch API deprecation

Multi-provider fallback (Adzuna, TheirStack, direct ATS). No single point of failure for job data.

High

EU AI Act non-compliance

Compliance checklist in progress, legal counsel planned, targeting August 2026 deadline.

Medium

Account bans (LinkedIn etc)

We don't automate browser actions. API-only approach eliminates ban risk entirely.

Medium

Claude API cost spikes

Usage caps per tier, aggressive caching, model fallback to Haiku for non-critical tasks.

Medium

Staffing agency ToS violations

Use JSearch (legal intermediary via Google for Jobs), not direct scraping.

Low

Competitor copies privacy angle

First-mover advantage, deeper implementation, EU compliance moat is hard to replicate.

Low

Supabase outage

Multi-region backup capability. Supabase 99.9% SLA on Team plan.
